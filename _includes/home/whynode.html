Why Nodes?
Since the launch of Bitcoin in 2009, the population of these specialized computers have grown faster than humans.
Most blockchain nodes only service a single network. Many require custom hardware, but Koii is lightweight enough to run on personal devices like phones and tablets, and the first network to achieve scalable generalized consensus.

Koii’s Layer-N network achieves this without its own blockchain— it uses all of them.

Each Koii node runs ‘Tasks,’ which have their own rewards and penalties for misbehavior. Tasks run asynchronously and can include writing to one or many blockchains.
