How it works

1. Download (and run) a node

A Koii node runs in the background on your computer, allowing you to be a part of the consensus process, completing KOII Tasks set by others.

2. Earn KOII

When you complete KOII Tasks, you are rewarded with KOII tokens. KOII can be spent to request other nodes to run Tasks, or traded.

3. Create Tasks, add to the Network

With your KOII, you can set bounties for Tasks, accomplishing your goal while contributing to the permanent archive.
